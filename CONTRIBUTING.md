# Contributing to CrownStar

Thank you for your interest in CrownStar! We welcome bug reports, feature requests, documentation improvements, and client SDK contributions.

## How to Contribute

1. **Open an issue** first to discuss the change you wish to make.
2. **Fork the repository** and create a branch for your contribution.
3. **Write clear, tested code** that follows the existing style.
4. **Submit a pull request** and link it to the relevant issue.

## Development Setup

- Clone the repo: `git clone https://github.com/AI-CrownStar-AI-BOT/CrownStar.git`
- Install Python 3.10+
- For any tier: `cd src/Free && pip install -r requirements.txt && python crownstar_desktop.py`

## Code of Conduct

Be respectful, inclusive, and constructive. CrownStar is for everyone.
