# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 3.x     | ✅        |
| < 3.0   | ❌        |

## Reporting a Vulnerability

If you discover a security vulnerability in CrownStar, please **do not** open a public issue. Instead, email the details to:

**skyocean2024@myyahoo.com**

We will respond within 72 hours and work with you to resolve the issue confidentially.

## Important Notes

- CrownStar does **not** collect or transmit any personal data.
- All internet harvesting is performed by the user’s own machine and only to generate responses.
- For any concerns about data privacy or security, contact us directly.
