# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 5.x     | ✅        |
| < 5.0   | ❌        |

## Reporting a Vulnerability

**Do not open public issues.**  
Email security reports to: `skyocean2024@myyahoo.com`

We will respond within 48 hours.

## Important Notes

- CrownStar runs entirely on your machine. It does not phone home or collect telemetry.
- Internet harvest (when enabled) sends requests from your IP – use a VPN if you require anonymity.
- No remote code execution. All amenities are local.

Thank you for helping keep CrownStar secure.
