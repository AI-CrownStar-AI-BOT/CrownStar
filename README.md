## 🧠 How It Works
1. You ask a question.
2. CrownStar harvests live internet (DNS, HTTP, Wikipedia, RSS).
3. Markov chain generates an answer.
4. If answer is shorter than minimum length, it gets ethically padded.
5. Everything is saved to SQLite – perfect recall.
