## 🧠 How It Works
1. You ask a question.
2. CrownStar harvests live internet (DNS, HTTP, Wikipedia, RSS) – up to 200 sources.
3. Markov chain generates an answer.
4. If the answer is shorter than your minimum length, it gets ethically padded.
5. Every detail is saved to SQLite – recall anything later.
