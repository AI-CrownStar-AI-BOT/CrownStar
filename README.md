## 🧠 How It Works – The CrownStar Cognitive Model

1. **Harvest** – Live internet data → raw text.
2. **Filter** – Gamma Burst selects most relevant sentences.
3. **Train** – Markov chain learns from filtered text.
4. **Generate** – Fluent, original answer.

For the mathematically inclined, see [MODEL.md](MODEL.md).
